How it works

You run the craft command

Choose the Ahoy that matches the language or framework your project is using and run the craft command.

We generate the files you need

Each language pack lives in a git repo. We fetch it and use it to generate the files your app needs for Docker.

Run your Docker environment

You've got all the files you need, all that's left is to spin up your new Docker development environment.

A little more about Ahoy...

Ahoy is an open source tool for preparing simple and lightweight Docker environments for your app.

It's sensibly opinionated so you can get you up and running as quickly as possible. You don't even need to know Docker to get started.

When you install a language pack, Ahoy will fetch the corresponding git repository. Contained in the repo is everything required to create fully functional Docker files for that language or framework.

Since everything is hosted in git repos, contributing your own language packs is incredibly easy and can great way to help other developers save a lot of time!

Ahoy is special because it's designed to be lightweight and built with deployment in mind. We won't bog you down with more than you need and you'll be able to extend with advanced functionality with ease when required.
